# PowerBI-retail-sales-dashboard
This project analyzes retail sales data using power bi.

Dashboard Overview

The dashboard provides insights into:

1.Total Sales

2.Total Profit

3.Total Orders

4.Total Quantity

5.Monthly Sales Trend

6.Sales by Category

7.Profit by Category

8.Top 10 Best-Selling Products

9.Regional Sales Distribution

Tools Used:

1.Power BI
2. DAX
3. Data Visualization
